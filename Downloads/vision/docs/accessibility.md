# Accessibility

SignBridge is built accessibility-first.

- All interactive elements meet WCAG 2.1 AA contrast.
- Keyboard navigation works everywhere.
- The `AccessibilityPreference` entity lets users enable:
  - `highContrast` mode
  - `largeText` mode
  - `reduceMotion` mode
- Captions for any rendered media are localised by `captionLanguage`.
- Sign language is configurable per user (`signLanguage` — ASL by default).
