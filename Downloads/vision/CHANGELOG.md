# Changelog

All notable changes to SignBridge are documented in this file.
The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added

- Initial production-ready monorepo scaffold.
- Express + Prisma backend with JWT, RBAC, refresh-token rotation, Zod
  validation, rate limiting, Helmet, CORS, structured logging.
- React + Vite + Tailwind frontend with auth flows, dashboard, sign-recognition
  workspace, translation studio, history, and admin panel.
- AI modules for MediaPipe Hands, OpenCV, and TensorFlow.js scaffolding.
- Vitest suite with coverage thresholds at 80%.
- OpenAPI 3.1 specification and Swagger UI.
- Docker, docker-compose, GitLab CI/CD, pre-commit, Husky, Gitleaks, git-cliff.
- Spec-Kit constitution, templates, and feature specifications.
